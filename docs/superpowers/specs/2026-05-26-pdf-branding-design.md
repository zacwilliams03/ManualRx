# PDF Branding Update — Design Spec

**Date:** 2026-05-26

## Problem

The prescription PDF uses a teal colour (`#3DBDB5`) that doesn't match the app's brand teal (`#29B5CC`), and has no ManualRx logo — the header just shows a plain clinic name string. Clients and patients receiving the PDF don't see consistent branding.

## Goal

- Align PDF colours to the app's brand palette
- Add the ManualRx logo to the PDF header
- Keep the document white/print-friendly (not dark)

## Scope

Single file: `src/components/therapist/PrescriptionPDF.jsx`

No changes to: `Prescribe.jsx` (call site), `pdfUtils.js`, routing, Supabase, or any other file.

---

## Design

### Colour Changes

| Element | Old value | New value |
|---|---|---|
| Teal accent (border, title, notes border) | `#3DBDB5` | `#29B5CC` |
| Teal highlight background (notes box) | `#E5F7F6` | `#E1F5FA` |
| Dark/navy text | `#1E2D3D` | `#1E2D3D` (unchanged) |
| Grey secondary text | `#6B7280` | `#6B7280` (unchanged) |
| Divider/border | `#D4E8E8` | `#D4E8E8` (unchanged) |
| Page background | white | white (unchanged) |

All colour values live in a `COLORS` constant object at the top of the file — only those two hex values change.

### Logo Component

Replace the current `clinicName` plain-text header with a ManualRx logo built from `@react-pdf/renderer` primitives. No image file required.

**Structure:**
```
Row (flexDirection: row, alignItems: center, gap: 8)
  ├── View  — teal bar: width 3, height 20, backgroundColor #29B5CC, borderRadius 2
  └── View  — text column
        ├── Text — "Manual" (color #1E2D3D) + "Rx" (color #29B5CC), bold, fontSize 16
        └── Text — "EXERCISE PROGRAM", fontSize 9, letterSpacing 1.5, color #6B7280
```

The `clinicName` prop is still accepted by `PrescriptionPDF` (no call-site changes needed) but is no longer rendered — the logo is always shown.

### PDF Structure After Change

```
┌─────────────────────────────────────────┐
│ ▌ ManualRx                              │  ← logo (teal bar + text)
│   EXERCISE PROGRAM                      │  ← subtitle (grey, small caps)
├─────────────────────────────────────────┤  ← 3px teal border
│ Client · Program · Date                 │
├─────────────────────────────────────────┤
│ 1. Exercise name                        │
│    3 sets × 10 reps · Bodyweight        │
│  ▌ Therapist note (teal highlight box)  │
├─────────────────────────────────────────┤
│ 2. Exercise name  …                     │
├─────────────────────────────────────────┤
│         Generated by ManualRx           │  ← footer (unchanged)
└─────────────────────────────────────────┘
```
